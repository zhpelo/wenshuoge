## 🍎 什么是公版书？

根据我国现行「著作权法」第 20、21 条的规定，除署名权、修改权、保护作品完整权外，中国公民对其著作的法定权利均于作者死亡后第五十年的 12 月 31 日截止。超过著作权法保护日期后，其作品就进入了公有领域（公共版权）

这种因作者死亡超过 50 年而丧失发行权、改编权等著作权利的书籍，就称为“公共版权书籍”，简称“公版书”。
